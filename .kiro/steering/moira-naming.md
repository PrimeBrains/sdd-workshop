# Moira 用語集（語彙の唯一所有者）

本ファイルが Moira の語彙の**唯一の所有者**である。他の steering は用語を再定義せず、ここを参照して使う。各語の正典は `moira/MODEL.md` の該当章（語源・神話は `moira/NAMING.md`）。

## 用語集

| 用語 | 意味 | 正典 |
|---|---|---|
| `EV_abs` | 絶対出来高 = Σ(完了サブ単位の凍結見積)、単位 MD | P1 / §3 |
| `EV%` | 達成率 = EV_abs / Σ(合意済み最新見積) ∈ [0,1]。無印「EV」は EV% を指す | P1 / R-U8 |
| 見積カバレッジ | Σ(合意済み見積ノード) / Σ(既知ツリー全ノード)。EV% と対で読む | P2 |
| スケジュールカバレッジ | 合意済みのうちスケジュール済みの割合。SPI と対で読む | R-S6 |
| `PV` | ベースライン予算（時間配分された計画値、MD） | §3 |
| `AC` | 実コスト = 自ノード cost + Σ子 AC。行為者に帰属 | P3 |
| `SPI` | = EV_abs / PV（スケジュール済み領域限定） | §3 / R-S6 |
| `CPI` | = EV_abs / AC | §3 |
| `c(i,d)` | 人間 i の暦日 d の日次容量 ∈ [0,1.0] MD/日（既定 1.0、理由付き） | A4 |
| `α_i` | 契約レート = c の reason=契約 成分の**派生 view**（一級パラメータではない） | A4 / §2.1#5 |
| `MD` | man-day = 人間アテンション時間。Moira の単一通貨 | A6 |
| `proposed` / `agreed` | 見積の状態。`agreed` への遷移は人間のみ | §2.2 / R-U4 |
| `implemented` | = 実装完了 ∧ 仕様FIX。**判定であって保証ではない** | §2.5 / P5 |
| `accepted` / `cancelled` | 完了（品質確認）/ 終端取消（どの非終端からも到達可） | §2.5 |
| supersede | 置換辺（新→旧）。エンハンスでの既存変更。**cancel とは別** | §2.7 / R-D7 |
| cancel | 終端取消。active basis から除外（サンク EV_abs は導出する） | R-C1 / R-C2 |
| 現行有効集合 | supersede されていない葉。累積 EV_abs と区別して読む | R-S5 / §2.7 |
| コミット領域 | 導出が語る対象（EV=合意済み・スケジュール=割当済み）。外は可視ギャップ | P0 |
| 4イベント | `transition` / `decompose` / `relate` / `cost` | §2.8 |
| at-risk | implemented 到達後に後退し未再到達のノードへの警告 | P5 |

## 三射影と名称（正典: `moira/NAMING.md §5`）

三女神が Moira の三射影に対応する: **クロト**=分解・計画（`decompose`）／ **ラケシス**=進捗測定・割当（EV・スケジュール）／ **アトロポス**=完了・取消（`accepted`/`cancelled`）。語源・神話の詳細は NAMING.md が正典。ダッシュボードの通称に使いうるが、**システム本体の名は Moira**。
