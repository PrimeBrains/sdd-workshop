# 思考実験 04–10 レビュー指摘事項台帳(FINDINGS)

本台帳は、思考実験 04–10 が MODEL v10(`moira/MODEL.md`)に対して露呈させた穴を、severity 付きで一覧化したものである。**MODEL 本体の修正は行っていない**——後続の `moira-model-update` スキル(独立敵対レビュー＝確定ゲート)を通して取り込む。本台帳はその入力。

## 自己検証プロトコルと確定結果

各思考実験は、サブエージェントによる**自己検証ループ**を通過した:
- **fact-checker**(`moira-fact-checker`):TE の MODEL 引用・行番号・§/要件コードを MODEL.md 現物と逐語照合。
- **反転 adversary ×3**(`moira-adversary`):通常と逆向き——TE が主張する各「穴」を**論破**しにいく(=MODEL が実はそのケースを処理している本文を引用できるか)。論破できない穴だけが finding として生存。立証責任は「穴は偽」を主張する側。
- **反転 gate-judge**(`moira-gate-judge`):falsifiable な PASS 基準=**(1) 誤引用ゼロ ∧ (2) 論理破綻ゼロ ∧ (3) 各 finding が adversary 過半の論破を生存し採点者の現物確認で追認 ∧ (4) severity が § 引用で根拠づけ**。

確定結果:**7 本すべて gate PASS**(誤引用 0・論理破綻 0・論破された偽穴 0)。生存した穴は **15 件(Critical 2 / Important 13)**。うち 2 件は設計者の意図に属する **FORK**(下記)。

ラウンド数:04(2R)・05(4R 収束)・06(1R)・07(2R)・08(2R)・09(3R)・10(1R)。

> **TE 05 の収束に関する注記:** TE 05 は当初「未合意完了で EV%>1/未定義」と主張したが R-U8(未合意を EV_abs から除外)で論破され、次に「モデルが処理する」と反転したが堅牢性確認モードで再論破され、最終的に**真の穴=半凍結ベースライン(PV/SPI/I4)**へ収束した。自己検証ループが偽穴を二度排除し本物へ到達した事例。

## FORK(設計者の意図に属する分岐) — ユーザー裁定済(2026-06-16)

露呈に徹する本作業では裁定しなかったが、ユーザーが解決の向きを確定した。実装は後続の `moira-model-update`(独立敵対レビューの確定ゲート)を通して行う。各 FORK の元の二読は記録として残す。

- **09-I2 → (A) スコープ外 + 矛盾警告。**(元: 意図的スコープ外説〔A4「能力をモデル化しない」の延長〕vs 未処理ケース説〔I6/R-U4 が権威境界を明示する以上その内側もモデルの関心〕)
  - 裁定: 人間内部の権威・オーナーシップは**モデル化せず**、誰が権威かは人間運用(P0)に委ねる。latest-wins(I3)が現行値を機械的に定める旨を明示する disclaimer を置く。**ただし「矛盾合意の検知 → 警告」は導出として持つ**——複数の人間が同一ノードに矛盾する `agreed` を並行発行したら、システムは異常として警告し人間が解決する(§0「観測・導出・警告に徹する」と一貫)。
  - 波及: この向きは **09-I1** も併せて解決する(競合裁定規則は追加せず、latest-wins + 矛盾警告 + 人間運用で閉じる)。`moira-model-update` の取込項目 = (1) 人間内部の権威はスコープ外という disclaimer、(2) **矛盾合意の警告**という新規導出/要件。
- **10-I2 → (A) 構造外(1 タスク 1 担当者)。**(元: 最小性原理〔§5「消費する UC が無いものは持たない」〕で構造外 vs 割当の多重度を構造化、の二択)
  - 裁定: ペア/モブ(一作業単位への同時複数人)は構造で持たず、**単一 assignee を意図的な前提として明示**する。理由: AI と人間の共同が一般化した現在、人間同士のペアプロはほぼ絶滅しており、消費する UC が無い(最小性 §5)。
  - 波及: この向きは **10-I1** も併せて解決する(単一 assignee の「暗黙の前提」を「意図的な仕様」へ昇格して明示)。cost 軸と計画軸の非対称(10-I2)は**設計どおり**となる。`moira-model-update` の取込項目 = 単一 assignee が意図的である旨の明示 + ペアを構造外とする注記。
