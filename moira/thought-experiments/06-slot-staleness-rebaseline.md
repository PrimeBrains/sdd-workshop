# 思考実験 06:凍結スロットの選定非決定性と陳腐化の不可視

> 問い:計画スロット(完了予定)は「初めてスケジュールに載った時」(§3 ②)に凍結される。だが平準化は発見的・非決定的(P8)である。**どの平準化実行の出力がスロットとして凍結されるか**は何が決めるのか。そして初回凍結後に割当が付け替わったら(§2.4)、凍結スロットは陳腐化しないのか。SPI = EV_abs/PV はその陳腐化したベースラインに対して測られないか。
>
> この思考実験は MODEL §3(導出指標、特に L166-172)と P8 の接合部に**穴を露呈させた**(MODEL 自体の修正は後の moira-model-update に委ねる。本書は露呈に徹する)。以下は MODEL v10 を前提とする。

## 素朴な期待と、それが破れた点

最初は「スロットは初回スケジュール時に凍結され、以後は正本として固定されるのだから、PV/SPI は安定して再現可能」と期待した。MODEL は確かにそう書く(§3 L167「記録後はその値が正本として固定され以後動かない」)。だが二つの点で期待が破れる。

1. **「凍結される値」がそもそも非決定的に選ばれる。** §3 L167 は明言する——「平準化は発見的・非決定的(P8)なので**どの完了予定が記録されるかはその一回の平準化に依存する**」「再生は**同一ログ・同一実装に対してのみ**保証」。つまり**正本性(L167「正本として固定」)は事後の不変性を与えるだけで、どの値を凍結するかの選定一意性は与えない**。
2. **凍結後の割当付け替えで前提が陳腐化しても、それを告げる導出が無い。** §2.4 L85 で割当は `(ts,id)` 順で付け替えられる。§3 L172 は「暫定割当の付け替え・取り消しはベースラインを動かさず生きた予測 P7/P8 だけを動かす」——ベースラインは**意図的に固定のまま**。だが固定されたスロットは、それを計算した割当が別人へ移った後も使われ続ける。

ここで重要な区別をしておく:**「ベースラインが固定であること」自体は欠陥ではない**(標準 EVM の PMB と同様、ベースラインは固定が正しい;§3 L173)。本実験が突くのはそこではなく、(1) **凍結値の選定手続きの非決定性**と、(2) **陳腐化を告げるシグナルの不在**である。

## 分析本体

### 綻び1:凍結スロットの選定が実装依存

§3 L166② は「計画スロットは…**初めてスケジュールに載った時…の完了予定で確定する**」とし、L168 は「どの『初回スケジュール』を基準とするか…は人間に委ねる」(早期暫定割当か `→implementing` 時か)。**だが「どの初回か」を人間が選んだ後も、その時点の平準化実行が出す完了予定は P8 ゆえ非決定的である。**

- L167 は非決定性を「除去するのではなく凍結する」と正直に書く。だが**凍結される一つの値をどう選ぶか**(同一時点でも複数の実行可能解がありうる平準化のうち、どの解の完了予定を正本にするか)の決定手続きは未規定。
- 帰結:**二つの正当な実装(あるいは同一実装の二回の実行)が、同一ログ・同一時点に対して異なるスロットを凍結し、異なる PV → 異なる SPI を「正本」として出しうる**。L167 の再現性保証は「同一実装に対してのみ」と明示的に条件付き。SPI は PM に示される健全性信号(§3 L171、R-S6)であるのに、その値が**モデルで決まらず実装で決まる**。

### 綻び2:陳腐化したベースラインに対する SPI に、対の読みが無い

MODEL は健全性指標を**必ず文脈と対で読む**規律を随所で課す:EV% は見積カバレッジと対(P2/R-S4)、SPI はスケジュール・カバレッジと対(§3 L171/R-S6)。ところが——

| 状況 | 起きること | 対の読み |
|---|---|---|
| 初回スケジュール時、暫定割当で slot 凍結(L168 が許す早期凍結) | PV にこのサブ単位の予算が当該スロットで載る | スケジュール・カバレッジ(R-S6)で「スケジュール済み割合」は読める |
| その後 `(ts,id)` で別人へ割当付け替え(§2.4 L85) | **凍結スロットは動かない(§3 L172)。だが生きた予測の完了予定は新割当で変わる** | **凍結スロットと現割当の乖離(=ベースライン陳腐化の度合い)を映す導出が無い** |
| SPI = EV_abs / PV(§3 L171)を読む | PV は陳腐化スロットに基づく。SPI は「もはや真でない計画」に対する進捗率 | R-S6 はスケジュール・カバレッジ(合意済みのうちスケジュール済みの割合)を見るが、**スケジュール済みのものが陳腐化しているかは見ない** |

§3 L168 は「早期に凍結した低情報スロットは**理由付き再ベースラインで更新できる**」と*可能性*を述べるが、**いつ更新すべきか(契機・義務)も、陳腐化を検知して人間に促す導出シグナルも規定しない**。EV%/SPI には「低カバレッジなら de-rate せよ」(R-S4/R-S6)という対の規律があるのに、**ベースライン陳腐化には対応する de-rate も可視のギャップも無い**。再ベースライン自体も綻び1と同じ非決定的平準化を再実行することになり、選定手続きの問題が再帰する。

## 見つかった穴

- **[Important] §3(L166②/L167)× P8(L162)** — 凍結スロットは非決定的平準化(P8)の一回の出力に依存する(L167)が、**どの実行可能解の完了予定を正本として凍結するかの選定手続きが未規定**。L167 の「正本として固定」は事後不変性であって選定一意性ではなく、再現性は「同一実装に対してのみ」(L167)条件付き。結果、PV→SPI が**モデルでなく実装で決まる**。SPI は PM への健全性信号(§3 L171/R-S6)なのに、その値が実装依存。/ V3(「正本」が一意性を含意するかの隠れ前提)・V4(スケール下で複数の実行可能解が分岐)。
- **[Important] §3(L168/L172)× R-S4/R-S6** — 初回凍結後に割当が付け替わる(§2.4 L85)とベースライン・スロットは陳腐化するが(§3 L172 で意図的に固定)、**陳腐化を検知・警告する導出も、SPI を de-rate する対の規律も無い**。L168 は「再ベースラインで更新できる」と可能性のみ述べ、契機・義務・シグナルを規定しない。EV%/SPI には対の読み(R-S4/R-S6)を課すモデルが、**ベースライン陳腐化だけ可視のギャップ(P0)を持たない**という非対称。/ V6(未処理ケース)。

これら二点を辿ると、PMB の二次元凍結(§3 の v10 是正点)が、**スロット次元で選定非決定性と陳腐化不可視という穴を残している**ことが分かり、**穴が見つかった**(網羅の証明ではない;MODEL §5・P2 と同じ留保)。

*The slot is frozen from one run of a non-deterministic leveling (P8); "authoritative once recorded" (L167) gives post-hoc immutability, not selection uniqueness, and reproducibility is conditional on "same implementation" — so PV→SPI is implementation-determined, not model-determined. And once frozen, reassignment (§2.4) staleness is invisible: unlike EV%/SPI (paired with coverage, R-S4/R-S6), baseline staleness has no de-rate, no signal, and no visible gap.*

## レビュー来歴

独立敵対レビュー(自己検証ループ:fact-checker + 反転 adversary×3 + gate-judge)を**第1ラウンドで一発通過**(誤引用0・論理破綻0・両 finding が論破を生存)。3 体の反転 adversary とも、凍結スロットの選定一意性を与える本文も、陳腐化を信号化/de-rate/再ベースライン契機を規定する本文も MODEL から引けず refuted=false(L167 はむしろスロットが実装依存であることを自認)。敵対者の「乖離を映す導出が無い/不可視」を言い過ぎとする Important 指摘は、操作的 finding 文が P0 の専門語義で正確(二端点表示 ≠ 可視のギャップ ≠ 乖離度の導出)と裁定され、偽穴の捏造に当たらず生存。
